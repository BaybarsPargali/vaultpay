## Technical Architecture

### CLI Bridge (Production Approach)

**This is the CORRECT production approach, not a workaround.**

The official `@solana-program/token-2022` JS SDK provides instruction builders but NO ZK proof generation.
Proofs can only be generated by:
- Rust crate: `spl-token-confidential-transfer-proof-generation`
- CLI: `spl-token` (wraps the Rust crate)

```bash
# Server requirements
cargo install spl-token-cli
export SOLANA_KEYPAIR_PATH=/path/to/keypair.json
export SOLANA_RPC_URL=https://devnet.helius-rpc.com/?api-key=...
```
